### What I learned
I learned how to center webpage content using CSS Flexbox.

```css
.wrap {
    min-width: 100%;
    min-height: 100%;
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    margin-bottom: -25px;
}
```
